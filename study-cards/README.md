# Study Cards

Минималистичный локальный тренажер вопросов и ответов.

## Как запустить

```bash
cd study-cards
npm start
```

Открой `http://localhost:8123`.

Для смысловой проверки ответов через Gemini добавь `GEMINI_API_KEY` в файл `.env` в корне репозитория. Пример переменных находится в `.env.example`. Файл `.env` нельзя добавлять в Git.

## Что уже есть

- добавление и редактирование карточек;
- тренировка в формате вопрос -> ответ;
- простая система повторений;
- локальное сохранение прогресса в браузере;
- импорт и экспорт карточек через JSON;
- фильтр по темам.
- отдельный раздел English vocabulary с флеш-картами слов;
- пример React + TypeScript компонента в `components/EnglishVocabularyFlashcards.tsx`.

## Формат импорта

```json
[
  {
    "topic": "Тема",
    "question": "Вопрос",
    "answer": "Ответ"
  }
]
```

Также поддерживаются ключи `q`/`a` вместо `question`/`answer`.
